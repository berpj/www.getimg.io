Parameters

NameDescriptionDefault value
img *URL to imagenone
widthResize image to fit this width (in pixels)none
heightResize image to fit this height (in pixels)none
qualitySet image quality (between 1, lowest image quality and highest compression, to 100, best quality but least effective compression)92 for JPEG, 75 for PNG
rotateRotate image (in degrees)0
flipFlip imagefalse
* : required parameter
